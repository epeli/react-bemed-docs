# Modifiers

## Boolean Mods

Most basic modifier is the boolean mod.

```tsx{3,3}
const Badge = bemed({
    mods: {
        important: true,
    },
})("Badge");
```

It just adds the `Badge--important` class when `important` is passed as the prop

```tsx
<Badge important />
// =>
<button class="Badge Badge--important" />
```

## String Mods

When a string is passed to the mod definition

```tsx{3,3}
const Badge = bemed({
    mods: {
        important: "really-important",
    },
})("Badge");
```

It will be passed as is when the prop is added

```tsx
<Button important />
// =>
<button class="Badge really-important" />
```

## CSS Mods

CSS mod is like the boolean mod but the given CSS is also injected when it's
used

```tsx{1,1}{6,6}
import { css } from "react-bemed/css";

const Badge = bemed({
    mods: {
        important: css`
            box-shadow: 0px 0px 30px 10px purple;
        `,
    },
})("Badge");
```

## Enum Mods

Enum mods should be used when only one mod value makes sense at once

```tsx
const Badge = bemed({
    mods: {
        prize: {
            gold: css`
                color: gold;
            `,
            silver: css`
                color: silver;
            `,
            bronze: css`
                color: bronze;
            `,
        },
    },
})("Badge");
```

```tsx
<Badge prize="gold" />
// ->
<div className="Badge Badge--gold" />
```

Values of the enum definitions can be any other mod types (boolean, string, css).

In TypeScript the type of the `prize` prop will be `"gold" | "silver" | "bronze" | undefined"`.
